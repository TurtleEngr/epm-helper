# epm-helpers
Helper scripts for use with the ESP Package Manager (EPM)

THe "ESP Package Manager (EPM)" can be found at:
[https://jimjag.github.io/epm/](https://jimjag.github.io/epm/)

For help type:

    man mkver.pl
    man patch-epm-list

Documentation can also be found at:
